# Anjali Khilari - Portfolio

A modern, responsive portfolio website showcasing Full Stack Java Development and Flutter Mobile App Development skills.

## 🚀 Features

### Modern Design
- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic content
- **Gradient Backgrounds**: Beautiful color schemes and visual effects

### Dynamic Functionality
- **Smooth Navigation**: Fixed navbar with smooth scrolling between sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Contact Form**: Interactive contact form with validation
- **Scroll Animations**: Elements fade in as you scroll
- **Typing Effect**: Dynamic typing animation for hero title

- **Full Stack Development**: Java, Spring Boot, MySQL, REST APIs

- **Tools & Technologies**: Git, GitHub, Android Studio, VS Code, Maven
